DFD 게임 부분을 그리는 중 발견한 특이사항.

1.  : 게임 종료 후, 게임 로비에서의 캐릭터가 갖는 선택값을 어떻게 설정할까?

   ① default 값으로 초기화한다.

   ② 게임에서 사용한 값을 사용
     에러 : 스킬을 다 사용하였을 때 에러 발생.
     해결 : 게임 시작 시, 유저가 갖고 있는 스킬이 아닐 경우 False 값을 줌. (error 코드를 통해 메시지 출력)

2. 스킬 테이블은, User ID를 통하여 Union을 한다.

3. '스킬 정보'와 '캐릭터 정보'는 게임과 같이 로컬에 저장하여, 서버를 통하지 않고 불러들인다.
 --> 차후 도감에서 해당 정보를 불러들일 때, 서버를 거치지 않도록 하기 위함이다. ( 바뀔 수 있음 )

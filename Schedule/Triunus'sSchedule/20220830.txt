1. 이미지에 마우스를 가져다 되면, 해당 이미지에 대한 설명이 나오도록 한다.
 -  - IPointerHandler, IDragHandler, IDropHandler를 사용.

2. 이미지 범위(크기) 안에 마우스가 존재하는 것을 인식

3. 인식하였다면, 아주 약간 이미지의 색이 변경.

4. 인식한 이미지의 특정 번호를 인식하여 전달

5. 특정 번호에 해당하는 텍스트를 가져온다.

6. 가로로 긴 직사각형 네모(Prefab)를 생성 후, 마우스가 인식했던 이미지의 중앙에 위치하도록 함. 또는 자식으로 등록

7. 해당 Prefab 안에 스킬에 대한 텍스트를 적는다.
 : 이미 존재하는 skill 이미지에 대한 image.name을 받아와 해당 스킬의 skillnumber를 파악한다.

8. 해당 skillnumber에 해당하는 번호를 skillsInfo.json에서 찾아서 네모 Prefab의 하위 텍스트 컴포넌트에 기록한다.



블로그에 정리할 것들.

1. Web Request와 Response 형식과 System.NET으로 해당 형식을 조정하는 방법.

2-1. 특정 객체에서 부모 및 자식 객체에 접근하는 방법.

2-2. 특정 객체를 특정 객체의 자식으로 등록하는 방법.

3. Texture2D 형식 Assest를 이용한 이미지 갱신 방법

4. Image 형식 Assest를 이용한 이미지 갱신 방법.

5. Prefab 생성 방법.

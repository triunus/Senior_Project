1. Character 메뉴 시작 시, 초기 설정.

2. 버튼 클릭 시, 서버랑 컨텍 -> user가 가지고 있는 character info를 리턴

3. key=value&key=value 형식으로 주고 받음.

4. 리스폰은 하나의 변수 or 하나의 배열 or 하나의 객체 처럼 받을 수 있음.

5. 전달 받은 characterList는 UserPossessedCharacterList.txt에 저장.
 (저장 안하고 사용만 할 수도 있음)

6. 전달 받은 List 값에 해당하는 Character Image Prefab 만들어 놓기.

7. List의 길이만큼, 왼쪽 칸 생성.

8. 각 칸 마다 list[i]에 해당하는 image 삽입.

9. 마우스가 해당 이미지를 가리키면, 이미지 테두리 색 조금 변경

10. 해당 이미지 클릭 시, 해당 이미지의 이름을 통해 해당 이미지 정보 인식

11. CharacterInfo.json 만들어 놓기.

12. 인식한 정보를 통해 오른쪽 패널에, Character에 대한 상세 설명 기제.

13. Save를 통해 LobbyData.txt의 값을 부분 변경한다.

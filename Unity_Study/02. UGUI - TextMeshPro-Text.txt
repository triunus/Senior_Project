 - 유니티에서 제공하는 UGUI를 화면에 출력하려면, Canvas GameObject의 자식으로 배치해야 한다.


1. CanvasRenderer Component
 - Canvas에 배치되어 화면에 보여지는 UI에 필수로 포함되는 컴포넌트이다.
 - TextMeshpro - Text (UI), Image, Raw Image 등이 있다.



2. TextMeshPro - Text  Component
 - 화면에 텍스트를 표시하는 UI 컴포넌트
 - "Button", "Dropdown", "InputField"와 같이 화면에 텍스트를 표시하는 모든 UI가 "TextMeshPro - Text" 컴포넌트가 포함된 게임오브젝트를 가지고 있다.
 - "Text" 컴포넌트는 Legacy로 곧 사용이 중지될 예정으로 텍스트를 포함하는 다른 UI도 "TextMeshPro"가 붙은 것을 대체하여 사용한다.
 - 최초에 생성할 때는 "TextMeshPro" Asset을 Import하기 때문에, TMP Importer View가 활성화된다. (Import하면 됨!)

 - TextMeshPro - Text 컴포넌트를 스크립트로 컨트롤할 때,
  TMPro Name Space를 추가해야 사용 가능하다.
  또한, TextMeshProUGUI을 컴포넌트 명으로 사용해야 한다.

※ TextMeshPro란?
 - 텍스트 렌더링 파이프라인으로 SDF(Singed distance Field)를 사용한다.
  --> 포인트 크기나 해상도가 변해도 텍스트를 깨짐없이 깨끗하게 랜더링 가능)
 - 최적화가 잘 되어 있어서, 로컬라이징이 필요한 경우 언어별로 SDF를 생성해 지정할 수 있다.
 - 기존 Text로 하기 어려운 커스터마이징이나 다양한 텍스트 효과를 사용할 수 있다.
  (유니티가 TextMeshPro를 인수하여 그냥 사용하면 됨. <-- 'Toggle' 오브젝트만 "Text" 컴포넌트를 사용하고 있다.

1) [Text Input] 
 - Text Input Box 속성
  : 화면에 출력될 텍스트를 타이핑하는 공간
 - Text Style 속성
  : 텍스트의 크기, 색상 등이 적용된 텍스트 스타일.

2) [Main Settings]
 - [Text Input]의 Text Input Box 속성에 적힌 텍스트에 다양한 효과를 적용시킬 때 사용한다.
 - Font Asset, material preset, Font Style, font Size, Auto Size, Vertex Color, Color Gradient, Spacing Options, Alignment 속성...

3) [Extra Setting]
 - 문자영역 간격, 서식 택스트, 상호작용 등의 옵션 제공.
 - Margins, Rich Text, Raycast, Sprite Asset, Style Sheet Asset 속성이 있다.

4) 서식 있는 텍스트 (Rich Text) 지원
 - HTML과 유상하게 태그를 사용해 폰트 스타일 변경이 가능하다.
 


3. TextMeshPro - Text 폰트 에셋
 - 기존으로 제공하는 폰트 에셋은 영어에만 적용된다.
 - 따라서 한글 폰트를 제공하고 싶다면, 따로 폰트 에셋을 만들어 지정해야 줘야 한다.
 - https://www.youtube.com/watch?v=S_Vd-EqT9rI, 6분 7초를 참고하자!
